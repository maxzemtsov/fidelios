# v0.0.19

> Released: 2026-04-04

## Highlights

- **Version update notifications** — FideliOS now checks npm for newer versions every 2 hours and displays a blue banner when an update is available, similar to Claude.app. The sidebar shows the current version prominently with a blue arrow indicator when updates exist. Opt out with `FIDELIOS_DISABLE_UPDATE_CHECK=true`.

## Improvements

- **Desktop sidebar collapse** — Added SidebarClose/SidebarOpen toggle button in the breadcrumb bar. Sidebar collapses to zero width with smooth transition, main content stretches to fill.
- **Kanban horizontal scroll** — Board view now scrolls horizontally via trackpad/touch. Hold Cmd + scroll wheel for mouse horizontal scroll.
- **Mobile Issues sticky toolbar** — The New Issue / Search / Filter toolbar now stays pinned below the breadcrumb bar on mobile instead of scrolling into the status bar area.
- **Auto-retry on Anthropic 500 errors** — Transient `500`, `Internal server error`, and `api_error` responses from Claude API now trigger automatic 30-second retry instead of failing the run.

## Fixes

- **Sidebar bottom bar visibility** — Documentation link, version, settings, and theme toggle now properly hide when sidebar is collapsed.
- **Kanban board overflow** — Removed black right padding artifact on board view.
- **Agent model IDs** — Fixed 3 agents with invalid `claude-haiku-4-6` model ID to correct `claude-haiku-4-5-20251001`.
