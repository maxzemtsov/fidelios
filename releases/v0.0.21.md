# v0.0.21

> Released: 2026-04-04

## Highlights

- **One-click CLI install** — When an environment check fails because a CLI is missing (Claude Code, Codex, Gemini CLI, etc.), the agent config form now shows the install command with an **Install** button. Clicking it runs the package manager command server-side and automatically re-runs the environment test on success. New `POST /companies/:companyId/adapters/:type/install-cli` endpoint handles the install and returns the test result in one round-trip.
- **Landing page overhaul** — New demo video with autoplay, Docker installer scripts, three-column install cards, PWA manifest, favicon package, and Open Graph meta tags. Mobile overflow fixed, light theme redesigned.
- **Site copy improvements** — README rewritten with subscription-safe positioning, one-click install scripts, shield.io install buttons, and a cleaner Quick Start section.

## Fixes

- **Runs tab layout** — Removed fixed max-width from Runs tab to match other tabs. Save/Cancel bar no longer shrinks the Runs tab content.

## Improvements

- **Adapter install commands** — `installCommand` field added to `ServerAdapterModule` and `AdapterEnvironmentTestResult` types. Populated for all local adapters: `claude-local` (`@anthropic-ai/claude-code`), `codex-local` (`@openai/codex`), `cursor-local`, `gemini-local` (`@google/gemini-cli`), `opencode-local` (`opencode-ai`), `hermes-local` (`hermes-agent`).
- **Demo video** — Hosted locally to avoid broken GitHub attachment URL; autoplay triggers 2 seconds after page load.
