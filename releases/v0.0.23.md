# v0.0.23

> Released: 2026-04-06

## Fixes

- **Missing migrations 0046–0048 on install** — Fixed `column companies.peak_hours does not exist` error on first run. The `@fideliosai/db` build script used `cp -r src/migrations dist/migrations`, which created a nested `dist/migrations/migrations/` subdirectory when `dist/migrations/` already existed from a prior build. Migrations 0046–0048 were silently placed in the wrong path and never applied. Fixed by adding `rm -rf dist/migrations` before the copy step.
