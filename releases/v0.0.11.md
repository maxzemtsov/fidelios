# v0.0.11

> Released: 2026-04-01

## Fixes

- **Agent `$AGENT_HOME` resolution** — All local adapters (Claude, Codex, Cursor, Gemini, OpenCode, Pi) now explicitly inject the `$AGENT_HOME` path into the instruction directive. Agents previously wrote memory and personal files to the instructions directory instead of their workspace home.
- **Agent instruction context** — Companion files (HEARTBEAT.md, SOUL.md, TOOLS.md) are now described as already loaded into agent context rather than as files to read from disk, preventing "file not found" errors on heartbeat startup.
- **Bash tool `description` rule** — Added TOOLS.md onboarding asset for CEO agents documenting that the `bash` tool requires a `description` parameter.
- **PARA directory pre-creation** — Agent workspace PARA structure (`memory/`, `life/projects/`, `life/areas/`, `life/resources/`, `life/archives/`) is now pre-created on every heartbeat startup, preventing agents from failing when writing memory on first run.
- **Attachment icon** — Restored the `Paperclip` icon (lucide-react) for file attachments in issue comments, issue detail, and new issue dialog. The icon had been accidentally replaced with the FideliOS planet icon during rebranding.
