# v0.0.45

> Released: 2026-05-17

Inline-editable heartbeat intervals on the Settings → Heartbeats page.

## Edit heartbeat intervals in place

[#73](https://github.com/fideliosai/fidelios/pull/73) — the Instance Settings → Heartbeats page lists every agent's timer-heartbeat interval across all companies. That value is now **click-to-edit**: click the interval, type a new number of seconds, press **Enter** to save (**Escape** cancels) — no trip to the agent's full configuration page.

Saving reuses the existing `PATCH /agents/:id` path (`runtimeConfig.heartbeat.intervalSec`) — the same one the page's enable/disable toggle uses — and refreshes the list on success.
