iwr -useb https://fidelios.nl/install.ps1 | iex
What does this install?

macOS

  • Homebrew (if not already installed)
  • Node.js via Homebrew
  • FideliOS CLI (npm install -g fidelios)

Linux

  • Docker Engine (if not already installed)
  • Pulls ghcr.io/maxzemtsov/fidelios:latest
  • Runs FideliOS container on port 3100

Windows

  • Docker Desktop (if not already installed)
  • Pulls ghcr.io/maxzemtsov/fidelios:latest
  • Runs FideliOS container on port 3100
