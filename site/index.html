Common questions

Will my data be safe?
Yes. FideliOS runs entirely on your own computer (or your own cloud VM if you prefer). There's no FideliOS cloud, no data upload, no telemetry. Your agents talk to Claude / OpenAI / etc. directly via your keys — we never see the content. Privacy details →
What if an agent makes a bad decision?
Three layers of guardrails: (1) approval gates — hiring new agents, publishing content, or spending over a threshold always ping you for a 👍 first; (2) budget caps — every agent has a hard spending limit per day/month; (3) full audit log — every action is recorded, so you can see exactly what happened and undo it.
Do I need coding skills?
No. If you can open Terminal (Cmd+Space, type "Terminal") and paste one line, you can install FideliOS. From then on everything happens in a web dashboard or Telegram. First time on a Mac? →
How much does it cost?
FideliOS itself is free and open source — no subscription, no "pro tier". You pay the AI provider you choose (Anthropic, OpenAI, Google) for their API usage. Typical side-project: $5–20 per month. A one-person content operation runs around $30–60/month. Set a cap so it never surprises you.
Can I really run this from my phone?
Yes. Two ways: (1) Telegram bot — your agents message you updates, you reply in chat to approve or adjust priorities; (2) mobile web dashboard — same UI as on your laptop, optimised for touch, works in any phone browser.
Can I cancel anytime?
There's nothing to cancel — FideliOS is open source and runs on your machine. To uninstall: fidelios uninstall. Your data stays in ~/.fidelios/ until you delete that too.
What happens when I close Terminal?
The installer asks: "Keep FideliOS running automatically, even after you close Terminal or restart?" Pick Yes — it sets up a background service so your AI team keeps working 24/7. If you said No, you can change your mind later: Keep FideliOS running →
Why not just use Anthropic's Claude Managed Agents?
Anthropic's new Managed Agents run single agents on Anthropic's infrastructure — you pay $0.08/session-hour on top of token costs, and your prompts + tool results go through their sandbox. Great for a solo developer building one agent at a time.

FideliOS is the company-shape of the same idea: 5-50 agents in an org chart, with budgets, approval gates, Telegram remote control, peak-hour throttling — all running on your Mac / VM. Session hours are free (it's your electricity). Data never leaves your machine. Mix-and-match Claude, OpenAI, Gemini, and local models in the same company.

Short answer: use Managed Agents for single agents on Anthropic's cloud. Use FideliOS when you want a self-hosted company of agents you control end-to-end.
Can I use Opus 4.7's advisor tool / task budgets / fast mode?
Yes. Every Claude agent in FideliOS exposes these Anthropic beta features as plain config fields:
  • effort: low / medium / high / max — trade speed for quality per-agent
  • betas: advisor-tool-2026-03-01 pairs Sonnet+Opus for 30-50% cheaper agentic runs; task-budgets-2026-03-13 lets Opus 4.7 self-pace
  • maxBudgetUsd: hard spend cap per run
  • fallbackModel: auto-switch if your primary model is overloaded
