What does this install?

macOS

  • Homebrew (if not already installed)
  • Node.js via Homebrew
  • FideliOS CLI (npm install -g fidelios)

Linux

  • Node.js LTS via nvm (in ~/.nvm, no sudo)
  • FideliOS CLI (npm install -g fidelios)
  • Runs FideliOS on port 3100

Windows

  • Node.js LTS via winget (or nvm-windows fallback)
  • FideliOS CLI (npm install -g fidelios)
  • Runs FideliOS on port 3100
How does it work?

FideliOS runs a control plane that coordinates autonomous AI agents. Each agent has a role, tools, and instructions. You define your company structure, assign tasks, and agents execute them — writing code, managing deployments, handling ops, and more.

Is it free?

FideliOS is open source and free to self-host. You bring your own LLM API keys. The platform itself has no license fees.
